1.GET /api/food/suggest/{text}:
    -Input: là 1 string.
    -Output: là 1 json gồm tên và id các món ăn mà trong đó chứa string Input.Max 50 món (có thể sẽ thay đổi sau)
2.GET /api/food/search/{text}:
    -Input: là 1 string.
    -Output: là 1 json gồm tất cả thông tin của món ăn mà trong đó tên món ăn chứa string Input. Không có
            max, có phân trang(4 food /trang).
3.GET /api/food/{int:id}
    -Input: Là 1 int.
    -Output: là 1 json gồm tất cả thông tin của món ăn có id input.
4.POST /api/food/upload
    request body:
    {
        "file": [excelFile.xlsx]
    }
    -Output: nếu trả về 200 và kèm thông báo success thì đã add thành công tất cả thông tin trong file excel
             nếu trả về 201 và kèm tên của những món ăn bị trùng trong file excel.
5.GET /api/food/getall
    -Output: là 1 json gồm tất cả thông tin của tất cả món ăn, có phân trang(8 food/ trang)
6.GET /api/food/download/dup
    -Output: gửi 1 file excel trong đó chứa nhưng món ăn bị được xem là bị trùng.
